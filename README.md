## 📖 About the Project

This project implements advanced data analysis to map rice production performance in East Java Province. Using monthly datasets ranging from 2018 to 2025, the project combines descriptive statistical techniques to summarize historical trends with time-series forecasting models to project future production conditions over the next decade.

---

## 🎯 Analysis Objectives

* **Production Mapping:** Identify regencies/cities with the highest consistent rice production contribution annually.
* **Spatial & Temporal Analysis:** Visualize data using heatmap rankings and geospatial maps.
* **Future Forecasting:** Compare the effectiveness of various forecasting models—including **ARIMA**, **SARIMAX**, and **Prophet**—to capture seasonal patterns in agricultural data.
